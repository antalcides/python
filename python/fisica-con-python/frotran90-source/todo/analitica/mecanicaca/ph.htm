Modos normales de oscilación

En este applet se muestra el concepto de modos de oscilación del mecanismo objeto de un problema sobre vibraciones. En general el sistema se mueve en torno a la posición de equilibrio mediante la superposición de dos modos, cuyas amplitudes puede variar (e incluso anular) mediante las barras verticales. Puede también variar las constantes elásticas de los resortes (ambas son iguales) mediante la barra superior o la velocidad con la inferior.
