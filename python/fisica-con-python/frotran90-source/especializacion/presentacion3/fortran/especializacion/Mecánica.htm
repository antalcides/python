Perturbaciones y Teoría de KAM

Asumamos que tenemos un sistema integrable y que generamos una perturbación, entonces tenemos

La pregunta es que pasa con la integrabilidad para pequeños valores de e. Por ejemplo, este problema es de real relevancia en sistemas gravitatorios, en que H es periodico en q.

Si las frecuencias del problema inicial son racionalmente dependientes, entonces se generan orbitas cerradas denominadas "resonant tori". Si las frecuencias son racionalmente independientes, entonces se generan "non-resonant tori". Es posible expandir, y resolver el problema, sucesivamente en ordenes de e. Podemos demostrar el siguiente teorema de KAM (Kolmogorov, Arnold y Moser).

Teorema KAM: Si las frecuencias del problema inicial son racionalmente independiente (con ciertas restricciones) entonces el sistema H tiene soluciones predominantemente cuasiperiodicas que no se diferencian mucho de las soluciones del sistema Ho. Casi todos los "non-resonant-tori" son deformados, pero muy poco. Asi que el sistema tiene "non-resonant-tori", donde las órbitas son densas.
