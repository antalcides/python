Notas para correr fortran en nuestro sistema Linux:

Un programa en fortran siempre tiene la terminación .for y como los programas en IDL se puede editar desde cualquier editor de unix (p.e. vi ó emacs).

A continuación un programa "tonto" de fortran. El chiste es que lo copies, lo compiles y lo corras para que veas como funciona. Nota que si una línea empieza con "c" son comentarios y no se toman en cuenta como intrucciones de fortran. Los comentarios sirven para hacer anotaciones dentro del programa. Escribe siempre anotaciones en tu programa para que siempre te acuerdes cómo funciona y para que otros puedan entenderlo.

xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
c  la primera instrucción en fortran es darle un título al programa
c  además es recomendable anotar datos como el autor y la fecha
         PROGRAM tonto
c       Americo Gonzalez
c       VERSION I.0         Enero'99
c ---------------------------------------------------------------------------
c   La siguiente instrucción es la declaración de las variables del programa

       CHARACTER*30 nombre

c --------------------------------------------------------------------------
c  después de declarar las variables comienza el programa.
c--------------------------------------------------------------------------
10    write(*,*)'  Dime como te llamas? '
        read(5,40) nombre
        write(*,*)'  Hola '  nombre
        write(*,*)'  '
        goto 10
40     FORMAT(A40)
        end
xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
ya que el programa "tonto.for" está escrito y salvado, el siguiente paso es compilarlo.
para compilarlo escribes:

ariel:>g77 -o tonto tonto.for

esta instrucción te va a generar un archivo tonto, el cual es un archivo ejecutable. Para correr el programa tonto, solamente escribe:

ariel:>tonto

A continuación otro ejemplo de un programa de fortran que abre un archivo de datos y saca promedios para escribirlos en otro archivo. Sigue las anotaciones para que entiendas cómo funciona este programa

xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
c  la primera instrucción en fortran es darle un título al programa
c  además es recomendable anotar datos como el autor y la fecha
c ---------------------------------------------------------------------------
        PROGRAM Promedios
c       Americo Gonzalez
c       VERSION I.0         Enero'99
c ---------------------------------------------------------------------------
c   La siguiente instrucción es la declaración de las variables del programa
c   nota que en este caso comienza con "implicit none" lo cual
c   significa que todas las variables deben ser declaradas
c  INTEGER: son variables de números enteros
c  REAL: variables de números reales
c  CHARACTER: variables no numéricas (palabras)
c--------------------------------------------------------------------------
c VARIABLES
c--------------------------------------------------------------------------
        implicit none
        INTEGER i,contador
        REAL columna1,columna2,columna3,tot_column1,tot_column2,tot_column3
     +           prom_columna1,prom_columna2,prom_columna3
c --------------------------------------------------------------------------
c  después de declarar las variables comienza el programa.
c  En este programa vamos a abrir dos archivos: el "datos.dat" de donde
c  vamos a calcular los promedios y el "resultados.dat" en donde vamos a
c  escribir los resultados
c---------------------------------------------------------------------------
        Open (unit=15,file='datos.dat',status='old',readonly)
        Open (unit=18,file='resultados.dat',status='new')
c
c  antes de comenzar las sumas, igualamos a cero las variables adecuadas
c
        tot_columna1 = 0
        tot_columna2 = 0
        tot_columna3 = 0
        contador=0
c
c  Supongamos que en este caso el archivo "datos.dat" tiene tres columnas
c  de datos de 100 renglones cada una
c  Ahora vamos a leer el archivo y sumar cada serie de datos
c
        Do    i=1,100
                Read(15,*) columna1, columna2, column3
                tot_columna1 = tot_columna1 + columna1
                tot_columna2 = tot_columna2 + columna2
                tot_columna3 = tot_columna3 + columna3
                contador = contador + 1
        EndDo
c
c Ahora vamos a sacar el promedio de cada columna
c
        prom_columna1 = tot_columna1 / contador
        prom_columna2 = tot_columna2 / contador
        prom_columna3 = tot_columna3 / contador
c
c  Ahora vamos a escribir los resultados en el archivo "resultados.dat"
c  usando el formato 50

          write (6,50) 'promedio columna 1 = ',  prom_columna1
          write (6,50) 'promedio columna 2 = ',  prom_columna2
          write (6,50) 'promedio columna 3 = ',  prom_columna3

c  explicación del formato:
c  2x = 2 espacios
c  A21 = serie de 21 caracteres (palabras)
c  F7.3  = real de siete caracteres, tomando tres dígitos despúes del punto decimal
c
50    format (2x,A21,F7.3)

c fin del programa

        END

ya que el programa "promedios.for" está escrito y salvado  el siguiente paso es compilarlo.
para compilarlo escribes:

ariel:>g77 -o promedios promedios.for

esta instrucción te va a generar un archivo promedios, el cual es un archivo ejecutable. Para correr el programa promedios, solamente escribe:

ariel:>promedios
