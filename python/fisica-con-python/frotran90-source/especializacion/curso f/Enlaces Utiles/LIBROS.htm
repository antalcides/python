LIBROS:

  • PROBLEMAS DE FISICA dirigido por S. Kosel Edit. MIR (contiene mas de 850 problemas de fisica elemental para entrenamientos de olimpiadas)

  • INTERNATIONAL PHYSICS OLIMPIADS volume 1 Editor: Wndemar Gorzkowski , institute of physics, Polish Academy of Sciences. Edit: World Scientific. (exelente compendio de problemas que utilizaron las selecciones mas importantes del mundo para entrenar a sus participantes rumbo a la olimpiada internacional, )

  • OLIMPIADAS DE FISICA DE LA UNION SOVIETICA , I. Sh. Slobodetski, V. A. Orlov, Edit MIR ( Ingeniosisimos problemas de fisica elemental, todos con soluciones, )

  • PROBLEMAS DE FISICA dirigido por O. Ya Savchenko, edit MIR (mas de 2000 problemas con solucion solo para aquellos que ELLOS consideran dificiles, apto para olimpicos con experiencia )

  • PROBLEMAS DE FISICA GENERAL I. E. Irodov, edit MIR, (Completisimo libro de fisica, este libro contiene temas un poco avanzados para la olimpiada pero igualmete solubles que ellos, apto para olimpicos con experiencia )

  • FISICA EN EJEMPLOS Y PROBLEMAS, E. Butikov, A. Bikov, A. Kondratiev, Edit MIR. ( exelente libro didactico, en el se resuelven los problemas de olimpiada dificiles paso a paso, como si un maestro te explicara, apto para todo olimpico)

  • PROBLEMAS SELECCIONADOS DE FISICA ELEMENTAL, B. B. Bujovtsev, V. D. Krivchenkov, G. Ya. Miakishev,I. M. Saraeva. edit. MIR, ( tipico libro de problemas de olimpiada, con mas de 800 problemas y sus soluciones, apto para todo olimpico)

  • PREGUNTAS Y PROBLEMAS DE FISICA, L. Tarasov, A. Tarasova edit. MIR ( Buen libro didactico para todo olimpico)

  • *OLIMPIADAS INTERNACIONALES DE FISICA, Editor: Fernando Vega Salamanca, Universidad Antonio Nariño Bogpta, 1991( Conpendio de los examenes de todas las olimpiadas de fisica hasta 1990 )
