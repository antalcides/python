Estas son todas las funciones intrínsecas de propósito general del FORTRAN 5.1. Para saber más sobre alguna de ellas consúltese el manual del compilador, Microsoft FORTRAN Reference. No obstante diremos algunas notas sobre ellas.

En 1er lugar se ve que muchas de las funciones están como repetidas, como la función para buscar un máximo. Esto se debe a que, dependiendo del tipo de datos que se envíen a la función, esta devolverá un resultado con una determinada precisión. Existen distintas funciones para dar mayor o menor precisión al resultado. Depende del programador la elección de una determinada función acorde con la precisión requerida. Personalmente siempre suelo usar las funciones genéricas (gen), que dan una buena precisión con los distintos tipos de datos. Si se desea más, habrá que elegir la que se adapte a los datos y requerimientos.

En 2º lugar, las funciones trigonométricas reciben y dan sus valores en radianes.

En 3er lugar debemos tener cuidado, si deseamos mantener la precisión, y las funciones forman parte de expresiones, de que la asignación se haga a una variable que pueda contener el resultado.
