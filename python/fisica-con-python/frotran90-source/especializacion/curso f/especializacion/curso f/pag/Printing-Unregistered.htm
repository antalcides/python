Printing with an Unregistered Scientific Notebook

If your Scientific Notebook has expired you will sometimes find that the option to print is greyed out. What you need to do is make the document read only

When you click on the document in the web browser and see the dialogue box

choose save and you will see a dialogue box like

In the box illustrated here, the target folder is Lectures which is located in a folder called Fall-2002 in my computer. Choose whatever folder is convenient to you.

Now open your local Windows Explorer and browse to the file.

Point at the file and click the RIGHT mouse button

Click on Properties

and click to place a check mark in the read-only position. Then click on Apply to accept your changes.

You are back at the Explorer screen. Double click on the document and it will open in Scientific Notebook and you will now be able to print it.
