Use with EEPIC

EEPIC is a macro package extending the picture environment of . If you have the EPIC or EEPIC macros, and your dvi translator supports the tpic \specials, then you can save memory. With EEPIC pictures, the plot.tex file will be smaller, will run much faster (and need much less memory), and the dvi file will be smaller. The quality of the output is about the same. If you change the source, you can generate some more interesting line styles.

To use EEPIC, set GNUPLOT's terminal type to eepic instead of latex, and use GNUPLOT as before. The line styles will change. Include the file plot.tex in your document as before, along with the document style options [epic,eepic].
